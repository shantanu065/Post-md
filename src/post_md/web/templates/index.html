Post_MD v0.1
Rapid MD trajectory analysis — PCA · clustering · RMSD/RMSF/Rg/SASA/H-bond
1

Systems

Register one or more trajectories you want to compare (e.g. WT, Mutant 1, Mutant 2). Overlay-style analyses (RMSD / RMSF / Rg / SASA / H-bond) plot all systems on one graph. Browse to a file on disk — uploads aren't required for files already on your machine.

2

Analyses

Tick every analysis you want, fill its parameters, then hit Run analysis. All ticked analyses run across every saved system in a single batch.

Info — atom / residue / bond / frame counts per system.
RMSD — per-frame RMSD vs a reference frame after Kabsch alignment.
Plot customisation
RMSF — per-residue (default) or per-atom fluctuation about the mean structure.
Plot customisation
Rg — mass-weighted radius of gyration per frame.
Plot customisation
SASA — Shrake-Rupley solvent-accessible surface area per frame.
Plot customisation
H-bond — per-frame H-bond count using geometric D-H-A criteria.
Plot customisation
PCA — Cartesian PCA per system (saved as pca[_label].npz).
Plot customisation
Cluster — K-means on each system's PCA projections (PCA must be ticked in the same batch, or already on disk).
Plot customisation
