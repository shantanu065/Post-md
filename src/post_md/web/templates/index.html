Post_MD

Rapid MD trajectory analysis — PCA, clustering, RMSD/RMSF/Rg, with multi-system overlay comparison.

1. Systems

Add one system per trajectory you want to compare (e.g. WT, Mutant 1, Mutant 2). Each gets its own topology + trajectory file. When more than one system is registered, RMSD / RMSF / Rg plots will overlay all of them on a single graph with a legend. PCA + Cluster run separately per system.

2. Analyses

Tick every analysis you want, fill in its parameters, then hit Run analysis once. All ticked analyses run across all systems in a single batch.

Info — atom / residue / bond / frame counts for system 1.
RMSD — per-frame RMSD vs a reference frame after Kabsch alignment.
Plot customisation
RMSF — per-residue (default) or per-atom fluctuation about the mean structure.
Plot customisation
Rg — mass-weighted radius of gyration per frame.
Plot customisation
SASA — Shrake-Rupley solvent-accessible surface area per frame.
Plot customisation
H-bond — per-frame H-bond count using geometric D-H-A criteria.
Plot customisation
PCA — Cartesian PCA per system (saved as pca[_label].npz).
Plot customisation
Cluster — K-means on each system's PCA projections (PCA must be ticked in the same batch, or already on disk).
