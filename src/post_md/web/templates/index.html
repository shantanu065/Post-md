Post_MD

Rapid MD trajectory analysis — PCA, clustering, RMSD/RMSF/Rg. Every option from the post-md CLI is on this page.

1. Provide topology + trajectory

Two options per file. Upload sends the file to this server in resumable chunks — restartable on network drops, but still slow for very large trajectories. Use file on disk just records the absolute path — no upload, no copy — ideal for multi-GB / TB files when the server can already see them on disk.

Topology  .prmtop / .parm7 (AMBER), .gro (GROMACS), .pdb
Trajectory  .nc / .mdcrd (AMBER), .xtc / .trr / .gro (GROMACS)
